Plataforma de e-commerce para uma marca autoral de arte têxtil. Catálogo curado, página de ateliê com identidade visual quente, encomendas personalizadas por formulário, checkout integrado ao Stripe e emissão automática de etiqueta via API do Melhor Envio. O painel admin reduz o pós-venda a um botão: a artista despacha e volta a criar.
