Rede social para cinéfilos brasileiros descobrirem, avaliarem e compartilharem filmes. Alternativa ao Letterboxd pensada para o mercado brasileiro, com catálogo completo via TMDB, listas personalizadas e feed social. Built with React Native para iOS e Android.
